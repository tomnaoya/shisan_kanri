# 資産管理システム - フロントエンド

## 概要
クリニック向け資産管理システムのフロントエンドSPAです。  
GitHub Pages でホスティングし、Render 上のバックエンド API と連携します。

## デプロイ手順 (GitHub Pages)

1. このリポジトリを GitHub に push
2. **Settings → Pages → Source** で `main` ブランチ / `/ (root)` を選択
3. 数分後に `https://<username>.github.io/<repo-name>/` で公開されます

## 初期設定

1. ブラウザでフロントエンドを開く
2. ログイン後、左メニューの **設定** を開く
3. **API接続先URL** にRenderのバックエンドURLを入力して保存  
   例: `https://asset-management-api.onrender.com`
4. ページをリロード

## 初期ログイン情報
- ユーザーID: `admin`
- パスワード: `admin`
